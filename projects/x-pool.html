Project Overview

Players enter immersive stylized pool arenas where they put their pool skills to the ultimate test. Each match is turn-based, with players using intuitive touch controls to aim, apply spin, and execute shots. Power-ups like extended aim lines and spin correction alter the strategic landscape, while cosmetic upgrades and environment themes allow for a personalized, immersive experience. In each match players stake stable tokens elevating the competitive aspects of the game, complete with leaderboard incentives and NFT-bound rewards enhancing engagement.

Key Features

  • Realistic physics-based pool gameplay
  • Global matchmaking system
  • Rotating tournaments with prizes
  • Leaderboards and ranking system
  • Customizable cues and tables
  • Social features and chat
